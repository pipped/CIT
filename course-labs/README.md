# CIT Course Lab Notes

This folder converts additional CIT lab PDFs into portfolio-readable lab notes. Instead of storing the PDFs by themselves, each lab page summarizes the screenshots, commands, configuration values, and troubleshooting lessons shown in the original documents.

## Labs

| Lab | Focus |
|---|---|
| [Apache CloudStack Pod Setup](apache-cloudstack-pod-setup.md) | Server wipe, ESXi installation, RAID initialization, UniFi Dream Machine Pro setup |
| [Lab 3: Windows and Linux Network Commands](lab-3-network-commands.md) | `whoami`, `systeminfo`, `ipconfig`, `ifconfig`, `nslookup`, `netstat`, `ping`, `tracert`, `traceroute`, ARP |
| [Lab 4: IPv4, Subnetting, and Connectivity](lab-4-ipv4-subnetting.md) | IPv4 properties, subnet ranges, static network settings, local LAN gateway behavior, IPv6 basics |
| [Lab 7: Remote Access and RRAS](lab-7-remote-access-rras.md) | Windows Server Remote Desktop, firewall settings, local users, RRAS, VPN concepts |

## Skills Represented

- Windows Server administration
- Linux command-line networking
- IPv4 addressing and subnetting
- VMware ESXi deployment
- RAID and physical server troubleshooting
- Remote Desktop and RRAS concepts
- Network appliance setup and documentation
